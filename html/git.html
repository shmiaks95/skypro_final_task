Git

  • git ls-files - переводится как "список файлов" и выводит файлы, которые в данный момент отслеживаются Git.
  • git status — показывает состояние рабочего каталога и индекса: какие файлы изменены, добавлены, а какие вообще не отслеживаются.
  • git add . — добавляет в индекс все изменённые и новые файлы в текущей директории.
  • git push — отправляет локальные коммиты в удалённый репозиторий.
  • git restore — отменяет изменения в рабочей директории или в индексе.
