Jenkins

  • list-jobs — выводит список всех заданий на сервере.
  • delete-job <имя> — безвозвратно удаляет задание.
  • list-queue — выводит очередь сборок.
  • help — выводит список всех доступных команд. Если указать конкретную команду после help (например, help build), покажет подробную справку по ней.
  • build <имя_задания> — запускает сборку задания.
  • who-am-i — показывает текущие учётные данные и права пользователя.
